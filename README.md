# Paradise Nursery - Online Plant Store

## Project Overview
This is a React-based e-commerce application for a plant nursery called "Paradise Nursery". It allows users to browse a collection of houseplants, add them to a shopping cart, and manage their orders.

## Features
- **Landing Page:** Professional welcome screen with a "Get Started" button.
- **Product List:** 30+ unique plants organized into categories (Air Purifying, Aromatic, Low Maintenance).
- **Redux Cart:** Fully functional shopping cart with item addition, removal, and quantity updates.
- **Dynamic Calculation:** Real-time calculation of total costs and item counts.

## Tech Stack
- React.js
- Redux Toolkit
- Vite
- CSS3
